# OneMount Guides

This directory contains comprehensive guides for different audiences.

## Guide Categories

### [User Guides](user/)
End-user documentation for installing, configuring, and using OneMount:
- Installation guides
- Quickstart tutorials
- Troubleshooting help

### [Developer Guides](developer/)
Documentation for contributors and developers:
- Development setup and workflow
- Coding standards and guidelines
- Infrastructure and tooling
- Error handling and logging patterns

### [AI Agent Guides](ai-agent/)
Instructions and rules for AI agents working on the codebase:
- Operational best practices
- Coding standards
- Testing conventions
- Documentation conventions
- Git conventions
